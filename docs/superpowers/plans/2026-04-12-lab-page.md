> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `/lab` page — six interactive science calculators backed by peer-reviewed research, each a self-contained bento card with inputs, computed output, effect size, and citation. Pure TypeScript math, no backend required.

**Architecture:** New `LabPage.vue` route, one composable per interactive tool (supplement card is static — no composable), shared `LabCard.vue` + `LabEvidenceBlock.vue` wrappers. All math is deterministic client-side computation mirroring the already-built Python modules in `research/`. No Pinia — nothing on this page requires cross-component shared state.

**Tech Stack:** Vue 3 + Composition API, TypeScript, Lucide icons, scoped CSS bento grid, Vitest.

**Science honesty policy:** Every output must show (a) the effect size with its unit, (b) the study population it came from, (c) individual variation caveat where the research flags one. This is enforced structurally via `LabEvidenceBlock.vue` — not left to per-card discipline. No calculator may emit a benefit score when its inputs are invalid.

```typescript
// src/composables/lab/useNapCalc.ts
import { ref, computed } from 'vue'

export type NapCalcInput = {
  hoursAwake: number
  sleepDebtMin: number
  currentHour: number   // injected — not read from clock inside pure fn
}

export type NapCalcResult = {
  duration: number
  durationLabel: string
  idealStart: string
  nightPenalty: number   // % night sleep efficiency lost (0 if before 3PM)
  boostMinutes: number   // alert boost in minutes
  inWindow: boolean      // true if 13:00–15:00
  coffeeNap: boolean
}

export function calculateNapRecommendation(input: NapCalcInput): NapCalcResult {
  const { hoursAwake, sleepDebtMin, currentHour } = input
  const inWindow = currentHour >= 13 && currentHour <= 15

  let duration = 26
  let durationLabel = '26 min (NASA sweet spot)'
  if (sleepDebtMin > 60)    { duration = 90; durationLabel = '90 min (full cycle — significant debt)' }
  else if (hoursAwake < 5)  { duration = 10; durationLabel = '10 min (too early for longer)' }

  const idealStart = currentHour < 13 ? '13:00' : currentHour > 15 ? 'now (expect some night impact)' : 'now'
  const nightPenalty = currentHour > 15 ? Math.round((currentHour - 15) * 4) : 0
  const boostMinutes = duration === 10 ? 155 : duration === 26 ? 180 : 480
  const coffeeNap = duration <= 26

  return { duration, durationLabel, idealStart, nightPenalty, boostMinutes, inWindow, coffeeNap }
}

export function useNapCalc() {
  const hoursAwake   = ref(6)
  const sleepDebtMin = ref(0)
  const currentHour  = ref(new Date().getHours())

  const result = computed(() =>
    calculateNapRecommendation({
      hoursAwake: hoursAwake.value,
      sleepDebtMin: sleepDebtMin.value,
      currentHour: currentHour.value,
    })
  )

  return { hoursAwake, sleepDebtMin, currentHour, result }
}
```

```typescript
// Core math
export function calculateAlcoholImpact(
  drinks: number,
  weightKg: number,
  sex: 'male' | 'female',
  sleepHour: number,
  hoursSinceDrinking: number
) {
  const r = sex === 'male' ? 0.68 : 0.55
  const bac = Math.max(0, (drinks * 14) / (weightKg * r * 10) - 0.015 * hoursSinceDrinking)

  // Pietilä 2018
  const hrvDropPct = drinks === 0 ? 0 : drinks <= 2 ? -9.3 : drinks <= 4 ? -24 : -39.2
  const remLossMin = Math.min(Math.round(drinks * 12), 45)

  // Hours until BAC < 0.01 (safe to sleep)
  const hoursToClear = Math.max(0, (bac - 0.01) / 0.015)
  const cutoffHour = Math.max(0, sleepHour - hoursToClear)

  return { bac, hrvDropPct, remLossMin, hoursToClear, cutoffHour }
}
```

```typescript
export const TECHNIQUES = {
  resonance: { bpm: 5.5, label: 'Resonance', baseBoost: 30, extendedExhale: true  },
  box:        { bpm: 4,   label: 'Box',       baseBoost: 20, extendedExhale: false },

export function calculateBreathworkResponse(
  technique: keyof typeof TECHNIQUES,
  durationMin: number
) {
  const tech = TECHNIQUES[technique]
  // Dose-response (Laborde 2022)
  const durationMult = durationMin <= 5 ? 0.5 : durationMin <= 10 ? 0.75 : durationMin <= 20 ? 1.0 : 1.15
  // Van Diest 2014: extended exhale +15% vs equal ratio
  const ratioBonus = tech.extendedExhale ? 1.15 : 1.0
  const rmssdBoost = Math.round(tech.baseBoost * durationMult * ratioBonus)
  const residualHours = durationMin <= 10 ? 1 : durationMin <= 20 ? 2 : 4
  return { rmssdBoost, residualHours, bpm: tech.bpm }
}
```

**Validation-first approach** — overnight windows are out of scope for v1 and must surface an error, not silently wrap:

```typescript
export function calculateMealWindow(
  firstMealHour: number,   // decimal, e.g. 8.0
  lastMealHour: number,    // decimal, e.g. 20.5
  sleepHour: number        // decimal, e.g. 23.0
) {
  const invalidWindow      = lastMealHour <= firstMealHour
  const invalidSleepOrder  = sleepHour <= lastMealHour
  if (invalidWindow || invalidSleepOrder) {
    return { valid: false, error: 'Use same-day times: first meal < last meal < sleep time' }
  }

  const windowHours    = lastMealHour - firstMealHour
  const hoursBeforeSleep = sleepHour - lastMealHour
  const earlyTRF       = firstMealHour <= 9 && lastMealHour <= 14

  const score =
    (windowHours <= 10 ? 40 : windowHours <= 12 ? 25 : windowHours <= 14 ? 10 : 0) +
    (firstMealHour >= 7 && firstMealHour <= 9 ? 30 : firstMealHour <= 11 ? 15 : 0) +
    (hoursBeforeSleep >= 3 ? 30 : hoursBeforeSleep >= 2 ? 15 : 0)

  const glucoseBenefit = windowHours <= 10 ? '+10–15% glucose tolerance (Manoogian 2022)' : null

  return { valid: true, windowHours, hoursBeforeSleep, score, earlyTRF, glucoseBenefit }
}
```

```typescript
export function calculateExercisePhaseShift(
  hour: number,
  chronotype: 'early' | 'mid' | 'late'
): { shiftMin: number; morningMetabolicBonus: boolean; label: string } {
  // Youngstedt 2019 PRC
  const baseShift =
    hour <= 8  ?  60 :   // morning: advance
    hour <= 16 ?  45 :   // afternoon: moderate advance
    hour <= 22 ? -45 :   // evening: delay
    0

  // Thomas 2020: late chronotypes get ~1.5x shift from morning exercise;
  // early chronotypes get ~0.5x (already advanced, less to shift)
  const multiplier =
    chronotype === 'late'  && hour <= 10 ? 1.5 :
    chronotype === 'early' && hour <= 10 ? 0.5 : 1.0

  const shiftMin = Math.round(baseShift * multiplier)
  const morningMetabolicBonus = hour <= 10  // Sato 2019 clock gene expression
  const label = shiftMin > 0
    ? `+${shiftMin} min advance`
    : shiftMin < 0
      ? `${shiftMin} min delay`
      : 'Minimal phase effect'

  return { shiftMin, morningMetabolicBonus, label }
}
```

```typescript
})

```

## Science Framing Notes (for implementers)

The goal is to be the opposite of a wellness app. Every output should feel like reading a lab report, not a horoscope:
- **Show the number and its unit:** `-24% HRV` not "significantly reduced HRV"
- **Show the population:** "n=4,098 free-living adults" not "research shows"
- **Show the variation:** "Individual response varies 2–5x" where the research flags it
- **Never emit a score on invalid inputs:** validate first, output second
- **Supplement caveats must be visible by default** — not collapsed, not in a tooltip
